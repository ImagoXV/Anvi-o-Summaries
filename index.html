For layers

The directory misc data layers contains TAB-delimited files for additional data stored under the following data group names for each sample/layer found in the merged database: default.

The default data group, which often is added by anvi'o automatically and contains important information, contained these keys: total_reads_mapped, num_SNVs_reported, num_INDELs_reported, total_reads_kept.
